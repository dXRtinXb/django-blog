# Django Weblog

A simple and beautiful weblog built with **Django** as part of my practice and learning during an online course.

This project was created to explore core web development concepts with Django, including models, views, templates, routing, and basic CRUD functionality.

---

## ✨ Features

- User-friendly blog interface
- Create, read, update, and delete posts
- Django admin integration
- Responsive layout
- Clean and structured codebase
- Great for learning Django fundamentals

---

## 🛠️ Built With

- **Python**
- **Django**
- **HTML**
- **CSS**
- **SQLite**
